# tic-tac-toe-game
An interactive Tic-Tac-Toe game developed in python using tkinter that allows users to play against another player or challenge the computer, showcasing game logic and programming fundamentals.

## Features

- Player vs Player mode
- Player vs Computer mode
- Win detection
- Tie detection
- Reset Game button
- Tkinter GUI

## Technologies Used

- Python
- Tkinter
- Random Module
