# tic-tac-toe-game
An interactive Tic-Tac-Toe game developed in python using tkinter that allows users to play against another player or challenge the computer, showcasing game logic and programming fundamentals.

## Features

- Player vs Player mode
- Player vs Computer mode
- Win detection
- Tie detection
- Reset Game button
- Tkinter GUI

## Technologies Used

- Python
- Tkinter
- Random Module

## How to Run

1. Clone this repository
2. Open the project folder
3. Run:
python tic-tac-toe.py

##Game Rules
Choose either Player vs Player or Player vs Computer mode.
Click on an empty cell to place your symbol.
Get three symbols in a row, column, or diagonal to win.
If all cells are filled and nobody wins, the game ends in a draw.
